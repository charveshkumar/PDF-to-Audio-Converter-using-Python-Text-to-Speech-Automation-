Title : PDF to Audio Converter using Python (Text-to-Speech Automation)
Project Description : Developed a Python-based PDF to Audio Converter that transforms textual content from PDF documents into speech output using Text-to-Speech (TTS) technology. The application extracts text from multi-page PDF files using the pypdf library and processes it efficiently. The extracted text is then converted into an audio file using the Google Text-to-Speech (gTTS) API, enabling users to listen to document content instead of reading it manually. The system is designed with a modular class-based approach, ensuring clean code and scalability. This project demonstrates practical implementation of file handling, third-party libraries, and automation concepts in Python.
Key Features
•  Extracts text from multi-page PDF documents 
•  Converts extracted text into audio (MP3 format) 
•  Supports automated end-to-end PDF to speech conversion 
•  Handles empty or non-readable PDFs with validation 
•  Modular and reusable class-based design 
•  Simple and efficient user execution
Technologies Used

•  Concepts: pypdf, gTTS (Google Text-to-Speech)
•  Libraries: pypdf, gTTS (Google Text-to-Speech)
